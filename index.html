Add Value - Group Demo - Conceptual

What is recursion?

Add Value - Group Demo - Technical

In mathematics, the factorial of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n. For example,

5! = 5 * 4 * 3 * 2 * 1 = 120.

Using recursion, create a function factorial that receives n and returns n!.

Based on: CodeWars: Recursion #1 - Factorial

Add Value - Group Demo - Problem Solving

There's a contagious infection making the rounds. It quickly becomes clear that there are two types of individuals:

Type A: Susceptible (75-90% of the population)

  • Contract the infection.
  • Are sick for several weeks before returning to health.
  • Are contagious only while sick.
  • Can be reinfected.

Type B: Carriers (10-25% of the population)

  • Contract the infection.
  • Never get symptoms.
  • Are permanently contagious.

Needless to say, this is causing a national workforce problem.

Thankfully, an immunization has been created. Unfortunately, while it cures Type B people (i.e. they will no longer be carriers) it kills Type A people.

How would you identify the Type B people as quickly as possible?

Add Value - Group Practice - Conceptual

What is the difference between == and ===?

Add Value - Group Practice - Technical

What will the result be for each of the following comparisons?

  1. 0 == "0"
  2. "" == "0"
  3. 0 == ""
  4. false == "0"
  5. null == undefined
  6. 0 === "0"
  7. " \t\r\n" == 0
  8. [0] === [0]
  9. {zero: 0} === {zero: 0}

Add Value - Group Practice - Problem Solving

Given two arrays of objects, how would you go about returning an array of equivalent objects that appear in both arrays.

For example, given:

  • const groupA = [ { id: 12, name: 'Alice' },
		 { id: 23, name: 'Bobby' },
		 { id: 34, name: 'Charley'} ]
  • const groupB = [ { id: 23, name: 'Bobby' },
		 { id: 45, name: 'Danny' },
		 { id: 67, name: 'Freddy'} ]

You would need to return: [{ id: 23, name: 'Bobby'}]

Add Value - Pair Practice

  1. What is hoisting?
  2. Give an example of a function that will be hoisted, and an example of a function that won't be hoisted.

  3. The sort function is supposed to set the global variable to 1 when it has been executed at least once but it does not.

    Can you figure out why?

    You should not modify the behavior of the sort function.

 
var sorted = 0;
function sort(array) {
   sorted = 1; 
   for(var i = 0; i < array.length; i++) { 
     array[i]=3 * array[i] + 2;
   } 
   var n=array.length, mid=Math.floor(n/2);
   array=array.slice(0, mid).concat(array.slice(mid+1)); 
   var sorted=false; 
   for(var i=array.length - 1; i>= 0 && ! sorted; i--) {
      sorted = true;
      for(var j = 0; j < i; j++) { 
         if ( array[j]> array[j+1] ) {
            swap(array, j, j+1);
            sorted = false;
   }}}
   return array;
}
